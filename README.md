## Description

This project was built using Node.js and NestJS, and uses MongoDB as its database with Mongoose as the ORM. It includes an API documentation generated by Swagger which can be accessed at /api/docs. The application is also Dockerized.

Requirements
Node.js Active LTS version
TypeScript
MongoDB
Mongoose
Swagger
Docker
Installation and Usage
To use this project, follow these steps:

Clone this repository to your local machine.
Install the required dependencies using npm install.
Start the MongoDB database.
Build the Docker images by running docker-compose build.
Start the Docker containers using docker-compose up.
Access the application at http://localhost:3000/api/docs.
Testing
To run tests for the server component, use the following command:

bash
Copy code
npm run test
This project has at least 30% test coverage (statements).
